# Hello_Unity

![테서렉트](https://64.media.tumblr.com/e114a1412a762e215ebd86a8790cb1aa/tumblr_pvbjzdgEbY1xoe44go3_r3_540.gifv)

**Unity Hub(관리자/매니저)** : Unity 프로젝트 및 설치를 탐색, 다운로드 및 관리하는 과정을 간소화하는 App

- 다양한 버전을 다운로드하고 관리할 수 있음

- 모든 프로젝트 목록을 보고, 프로젝트에 맞는 에디터 버전을 연결하며, 클라우드/로컬 프로젝트를 관리

- 에디터 설치 시 필요한 Android SDK, Visual Studio 등 부가 모듈을 선택 설치 가눙

**Unity Editor(실제 작업자/엔진)** : 3D 또는 2D 세계를 구축하고 인터랙티브 애플리케이션을 제작하는 데 사용하는 App

- 게임 오브젝트 배치, 애니메이션, 물리 효과 적용 등 모든 개발 작업은 여기서

- 여러 버전(예: 2022 LTS, 2023, Unity 6 등)을 설치할 수 있으며, 각 버전마다 기능과 안정성이 다름
